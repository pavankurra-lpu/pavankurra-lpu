### 🛣️ NHAI Smart Highway Mobility & Toll Intelligence Platform

> A premium, portfolio-grade Indian highway mobility simulation inspired by real NHAI and FASTag workflows — featuring a complete **Dual-Portal System** for both Travellers and Operations Administrators.

<details>
<summary><b>🟢 User Portal — Traveller End</b></summary>

- 🗺️ **All-India Route Planning** via interactive Leaflet.js map (Standard / Dark / Satellite layers)
- ⛅ **Live Weather Intelligence** — dynamic engine tracks origin/destination weather impact on ETAs
- 🔀 **Smart Route Engine** — Fastest, Lowest Toll & Balanced routes powered by Dijkstra's algorithm
- 🚗 **NHAI Vehicle Categories** — LMV / LCV / HCM and pre-registered priority vehicles
- 💳 **FASTag Wallet Simulation** — balance checks, 3rd-party recharges with 1% fee calculation
- 🎫 **Toll Pass Intelligence** — Monthly / Trip / Annual pass recommendations per route
- 🚦 **Lane Allocation** — Live entry/exit lane indicators with priority lane support
- 🆘 **Emergency Reporting** — SOS incident dispatch (breakdowns, accidents, health) to Admin portal

</details>

<details>
<summary><b>🔴 Admin Portal — Operations End</b></summary>

- 🔐 **Secure Login** — protected access for NHAI operations staff
- 📊 **Live Operations Dashboard** — real-time overview of active highway issues & vehicle counts
- 📹 **CCTV Surveillance Simulation** — animated feed tiles for lane & gate monitoring
- 🚨 **Incident Response System** — acknowledge and resolve user-reported emergencies
- 📢 **Traffic Alert Broadcaster** — push live warnings (accidents, congestion, weather) to Traveller portal
- 🚑 **VIP & Emergency Vehicle Analytics** — track special passage requests in real-time

</details>

**Architecture Highlights:**
- 🏗️ Client-side distributed system — cross-tab real-time sync via `localStorage` events (mimics WebSocket behavior)
- 📐 Custom weighted graph + Dijkstra's algorithm powering multi-modal route optimization
- 🌓 Automatic Dark / Light Mode with persistent global state
- ⚡ Zero dependencies beyond Leaflet.js — pure HTML5 / CSS3 / Vanilla JS ES6+

`HTML5` `CSS3` `Vanilla JS (ES6+)` `Leaflet.js` `Dijkstra's Algorithm` `localStorage` `sessionStorage` `IoT-Inspired`
